Tor: Contribute

Users:

  • Try Tor out, and let us know about bugs you find or features you don't find.
  • Please consider running a server to help us with development and scalability.
  • Does somebody want to help maintain this website, or help with documentation, or help with managing our TODO and handling bug reports?
  • Please fix up the FAQ, and if you know the answer to a question in the "unanswered FAQs" list, please answer it.

Graphics folks:

  • We need a Tor logo.
  • We need a snazzy diagram or two, akin to the one BitTorrent has in its introduction, to show people how Tor works.

People with sysadmin skills:

  • We need somebody to help out with our RPM spec file (there's an in-progress version in the tarball).
  • We could use an OS X package too.
  • Can somebody take a look at Martin's Squid and Tor page, and update it to reflect Tor's new RedirectExit config option?

Developers:

  • We need somebody to code up a GUI or other controller program, to do configuration, etc. See our control specification for details, and the rudimentary python control script.
  • We're always looking for better Windows installers.
  • Does somebody want to do up a patch so we can be an NT service? Or so we can go in the system tray?

People with money:

  • Go take a look at the Electronic Frontier Foundation.

Let us know if you want to help out!

Tor: Pakete und Quellcode

Tor wird als Freie Software unter der 3-clause BSD license verteilt.

Die letzte Version kannst du aus dem Downloadverzeichnis beziehen. Die letzte Test Version ist 0.1.1.4-alpha. Tor sollte auf Linux, BSD, OS X, Win32, Solaris und anderen Systemen laufen.

  • Windows Installationsprogramm: 0.1.0.13 (sig), 0.1.1.4-alpha (sig). Stelle sicher, dass du die Win32-spezifischen Anweisungen gelesen hast.
  • Mac OS X Installationsprogramm: 0.1.0.13 (sig), 0.1.1.4-alpha (sig). Lies die OS X spezifischen Anweisungen.
  • Red Hat Linux Paket: 0.1.0.13 RPM (sig), 0.1.0.13 SRPM (sig), 0.1.1.4-alpha RPM (sig), 0.1.1.4-alpha SRPM (sig)
  • Letzte Quellpaket: 0.1.0.13 (sig), 0.1.1.4-alpha (sig). Weiterhin wirst du libevent benötigen. Wenn du mutig genug bist, um die Quellen auf Windows zu bauen, wirst du libevent 1.0c oder höher brauchen.

Allgemeine Anweisungen zur Installation und Konfiguration befinden sich hier. Der Eintrag in der FAQ beschreibt, wie die Signaturen der Pakete zu verifizieren sind.

Schaue dir die Entwicklerseite an, um Anweisungen zu erhalten, wie man Tor aus dem CVS-Verzeichnis erhält.

Ältere Versionen sind hier.

Debianpakete apt-get install tor (Hinweise zu Sarge und experimentellen Torversionen)

FreeBSD: portinstall -s security/tor

OpenBSD: cd /usr/ports/net/tor && make && make install (Anleitung für chroot)

NetBSD: cd /usr/pkgsrc/net/tor && make install

Gentoo: emerge tor (Anleitung)

Es gibt Gerüchte, dass andere Pakete für andere Plattformen existieren. Wenn jemand Details schickt, werden wir hier Links veröffentlichen.

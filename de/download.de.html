Tor: Pakete und Quellcode

Tor wird als Freie Software unter der 3-clause BSD license verteilt.

Die letzte Version kannst du aus dem Downloadverzeichnis beziehen. Die letzte stabile Version ist 0.1.0.14. Die letzte Test Version ist 0.1.1.6-alpha. Tor sollte auf Linux, BSD, OS X, Win32, Solaris und anderen Systemen laufen.

Im CVS findest du Anweisungen zur Installation und Konfiguration. Der Eintrag in der FAQ beschreibt, wie die Signaturen der Pakete zu verifizieren sind.

Sicherheitshinweise und neue stabile Versionen werden auf der Mailingliste or-announce bekannt gegeben. Wenn du Tor nutzt, solltest du ein Auge auf das Archiv werfen oder dich hier einzutragen: (Du erhälst eine Rückbestätigung per E-Mail.)

Tor sollte auf Linux, BSD, OS X, Windows, Solaris und anderen Systemen laufen.

  • Windows Installationsprogramm: 0.1.0.14 (sig), 0.1.1.6-alpha (sig). Stelle sicher, dass du die Anweisungen für Windows gelesen hast.
  • Mac OS X Installationsprogramm: 0.1.0.14 (sig), 0.1.1.6-alpha (sig). Lies die Anweisungen für OS X.
  • Debianpakete: apt-get install tor (Anweisung für Sarge und experimentelle Torversionen)
  • Red Hat Linux Paket: 0.1.0.14 RPM (sig), 0.1.0.14 SRPM (sig), 0.1.1.6-alpha RPM (sig), 0.1.1.6-alpha SRPM (sig)
  • Gentoo: emerge tor (Anweisungen)
  • FreeBSD: portinstall -s security/tor
  • OpenBSD: cd /usr/ports/net/tor && make && make install (Anweisung für chroot)
  • NetBSD: cd /usr/pkgsrc/net/tor && make install
  • Letzte Quellpaket: 0.1.0.14 (sig), 0.1.1.6-alpha (sig).

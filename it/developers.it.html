Tor: Sviluppatori

Sfoglia il reporitorio CVS di Tor: (che non necessariamente è funzionante)

  • cvs sandbox regolarmente aggiornato
  • Vai al CVS
  • accesso anonimo al pserver (la password è guest):
    • Crea una cartella vuota ed entraci.
    • cvs -d :pserver:guest@cvs.seul.org:/home/or/cvsroot login
    • cvs -d :pserver:guest@cvs.seul.org:/home/or/cvsroot co tor
    • cd tor; ./autogen.sh; make; make install se ti va.
    • (use "co -r tor-0_1_1_4_alpha tor" to fetch a particular version.)
    • (To check out the maintenance branch, use -r tor-0_1_0-patches)
